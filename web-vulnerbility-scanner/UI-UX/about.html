About DarkScanner

← Back

This scanner crawls a target website, discovers forms and URLs, injects payloads, and identifies likely vulnerabilities for XSS, SQL injection, and CSRF.

  • Uses requests and BeautifulSoup for crawling and input discovery.
  • Detects reflected XSS by payload reflection in responses.
  • Detects SQLi by injecting payloads and observing database error patterns.
  • Detects missing CSRF protection on POST forms.

Designed for education and authorized security testing only. Always get permission before scanning a live website.
